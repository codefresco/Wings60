# Wings60 Firmware

This directory contains both source and precompiled firmware for the Wings60 keyboard.

- Source code is in [wings60](wings60) if you want to modify or compile firmware yourself. See [wings60/readme.md](wings60/readme.md) for details.
- Compiled firmware is available in [release](release), ready to flash onto your keyboard.
